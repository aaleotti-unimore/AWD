Home

Pagina principale del sito, mostra una panoramica dei progetti per ogni utente (l'admin li visualizzerà tutti, in ordine di inserimento). Sono presenti 5 Bottoni funzione:

  • Description: mostra una descrizione del progetto, inserita da utente
  • Results: Mostra una pagina con i risultati elaborati da Matlabl assieme al codice sorgente del progetto
  • Launch: lancia un progetto, lasciando in attesa l'utente fino a quando non è completato (o per un massimo di {{ WATCHDOG_TIMEOUT }} secondi
  • Edit: Permette di modificare nome, descrizione e codice di un progetto
  • Delete: Permette di cancellare un progetto ed i suoi risultati. i files rimarranno nelle cartelle in cui sono stati salvati: {{ MEDIA_ROOT }}

Dal menu contestuale è possibile accedere ad ulteriori funzioni, spiegate di seguito.

Menu Projects: Create

Pagina di creazione progetto. è possibile accedere all'editor progetti oppure inserire direttamente un file .txt contenente il codice per completare immediatamente la creazione

Editor progetti

Permette di creare in maniera user-friendly un progetto. è possibile definire

  • Nodi tra cui ogni blocco viene definito indicando nome e coordinate cartesiane del nodo.
  • Blocchi del modello, ciascuno deve essere definito tra due nodi indicati precedentemente. Inoltre è possibile dare un nome personalizzato ad ognuna delle 4 variabili interne e indicare delle opzioni personali per ogni blocco premendo il tasto + a fianco di ogni riga di blocco.
  • Variabil di Sistema che infuluenzeranno l'intero progetto. nel testo placeholder è indicato il range in cui le variabili possono essere definite. uscire dal range causerà errore nei risultati

Manuale Admin

Menu Utente: Update Commands

Pagina per l'aggiornamento dei comandi. I file vanno caricati nelle corrette voci di input. i file devono contere obbligatoriamente i seguenti campi, anche vuoti:

  • Branch - Rami: Nome, Value, Sigla, StrNum, Vincoli, Range, Type, Help, Help_ENG
  • Block - Blocchi: Sigla, Tipo_di_Ramo, Diretto, Out, E_name, K_name, Q_name, F_name, Help, Comandi, Help_ENG
  • System - Sistema: Nome, Value, Sigla, StrNum, Vincoli, Range, Type, Help, Help_ENG

Una volta caricati saranno disponibili nell'Editor progetti e nella Command List dal menu Projects

Menu Utente: Admin

Pagia di amministrazione del database. Da qui è possibile fare tutte le modifiche manuali alle voci presenti nel database:

  • Gruppi: ignorare, non utilizzato dal sito
  • Utenti: Modifiche agli utenti.
  • Project Outputs: Pagina di modifica degli output progetto. ogni voce corrisponde ad un file di output i quasi sono assegnati ad un solo progetto. All'interno è possibile modificare i file di output presenti ed il progetto assegnato si raccomanda attenzione nell'inserimento di immagini e testo nei campi corretti o si renderà impossibile la visualizzazione dei risultati
  • Projects: Pagina di modifica progetti, da qui è possibile modificare nome, descrizione ed assegnare un differente file di codice. è possibile inoltre assegnare un progetto come "esempio" per gli utenti, aggiungendo la spunta ad "is_example"
